# Smartcane
Dự án Smart Cane là một thiết bị hỗ trợ di chuyển thông minh, tích hợp các công nghệ nhúng tiên tiến để cải thiện chất lượng cuộc sống cho người khiếm thị và người cao tuổi. Hệ thống được xây dựng trên nền tảng vi điều khiển ESP32 WROOM với kiến trúc đa nhân, kết hợp khả năng định vị vệ tinh, kết nối mạng 4G và tương tác bằng giọng nói AI.
